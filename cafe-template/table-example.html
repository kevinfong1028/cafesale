設計特性

📱 完全響應式

在桌面上顯示完整表格，在手機上自動轉換為卡片式布局。

🎨 設計一致

使用 Organic Naturalism 設計風格，按鈕顏色與品牌保持一致。

✨ 互動效果

行懸停效果、按鈕動畫，提升用戶體驗。

🎯 易於使用

清晰的按鈕標籤，在所有設備上都易於點擊。

🔧 易於自訂

使用 CSS 變數和 Bootstrap 類名，輕鬆自訂樣式。

📊 多用途

適用於產品列表、訂單管理、用戶管理等場景。

響應式斷點

螢幕寬度 表格行為
< 576px 卡片式布局，每行一個卡片，按鈕堆疊
576px - 768px 表格縮小字體，按鈕並排
768px - 992px 表格正常顯示，略微縮小
> 992px 完整表格，全尺寸顯示
