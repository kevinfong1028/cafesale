分頁特性

📱

完全響應式

在所有螢幕尺寸上都能完美顯示，自動調整間距和字體大小。

🎨

設計一致

使用 Organic Naturalism 設計風格，與整個網站保持視覺一致。

✨

互動效果

懸停動畫、活躍狀態指示，提升用戶體驗。

🎯

易於使用

清晰的按鈕標籤，足夠大的點擊區域。

🔧

易於自訂

使用 CSS 變數，輕鬆自訂顏色和尺寸。

♿

無障礙設計

支持鍵盤導航，符合 WCAG 標準。

使用建議

✅ 最佳做法

  • • 在表格或列表下方放置分頁
  • • 顯示當前頁碼和總頁數
  • • 禁用第一頁的上一頁按鈕
  • • 禁用最後一頁的下一頁按鈕
  • • 頁碼超過 5 個時使用省略號

⚠️ 常見錯誤

  • • 按鈕太小，難以點擊
  • • 沒有清晰的活躍狀態指示
  • • 顯示過多頁碼（超過 10 個）
  • • 在手機上不響應式
  • • 沒有禁用不可用的按鈕
